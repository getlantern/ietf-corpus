  $(go env GOPATH)/bin/ietf-mcp -- --corpus $HOME/code/ietf-corpus

What you can do with it

  • "Find every RFC that updates TLS 1.2, sorted by year." Code-mode uses get_document → updated_by → get_documents in one script.
  • "List all wire-format definitions in QUIC." search_elements with kind=wire-format, document=rfc-9000.
  • "What's the obsolescence chain back to RFC 793?" find_related walking obsoletes until empty.
  • "Which RFCs introduce IANA registries about HTTP?" search_elements with kind=registry, topic=http.
